# Persona-Trigger & Beweis-Sprache (Quelle: Projekt-Brief Section 6)

ICP: Kommunen **10.000-50.000 EW**, faktisch **BW + Bayern**, >10 Bestandskunden.

**Trigger-Reihenfolge (immer einhalten):**
Pflicht-Bindung -> Frist -> Konsequenz -> Foerderhebel -> Klima (Bonus).

**Top-Einwand:** "Kein Budget/Personal" -> Foerdermittel-Begleitung inkl., Eigenanteil <= X %.

| Persona | Anteil | Trigger-Sprache | Beweis-Sprache |
|---|---|---|---|
| Klima-/Energiemanager:in | ~60 % | KWP, EnEfG, Klimaneutralitaet, CO2-Bilanz, Foerderquote | abgerufene Foerdermittel, CO2-Reduktion, gesparte Stunden |
| Bauamt | ~30 %, wachsend | GEG, GModG, Sanierungsfahrplan, Substanzerhalt | Substanzerhalt, Sanierungspriorisierung (**nicht** Klimaziel) |
| Buergermeister:in | klein | Pflicht-Frist, politisches Risiko | Peer-Kommunen, Presse-faehige Cases |
| Kaemmerei | mitentscheidend | Foerderquote, Eigenanteil, Foerdermitnahme, AfA | konkrete Eigenanteils-Zahlen |

## Stage-spezifische Aufhaenger-Bausteine (BY-Anruf-Kaskade)

- **Close-Queue (Angebot versendet):** Bezug aufs konkrete Angebot, Rueckfragen aufloesen, Entscheidungsweg/Gremientermin klaeren.
- **Reach-out (Neu/Webinar, Versucht zu erreichen, Im Gespraech):** Anknuepfung ans Webinar-Thema, konkreter naechster Schritt (Expertengespraech 30 Min).
- **Daten-Chase (Daten angefragt):** Excel-Ruecklauf freundlich anmahnen, Hilfe beim Ausfuellen anbieten, Termin zum gemeinsamen Durchgehen.

## Konkrete Deliverables statt "Beratung"
beschlussreife Wirtschaftlichkeitsberechnung, gremienfaehige Vorlage, Vergabeunterlagen (UVgO/VOB).
